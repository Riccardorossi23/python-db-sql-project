# 📊 SQL + Python Sales Analysis Project

## 🧠 Description
This project simulates a small e-commerce database and performs data analysis using SQL and Python.

## 🛠️ Tech Stack
- PostgreSQL
- Python
- Pandas
- Psycopg2

## 📁 Structure
- SQL scripts for database creation and queries
- Python script for analysis

## 📊 Features
- Customer analysis
- Sales analysis
- Product performance
- Ranking customers

## 🚀 How to run
1. Create PostgreSQL database: sales_analysis
2. Run SQL scripts
3. Install dependencies:
   pip install -r requirements.txt
4. Run:
   python main.py
